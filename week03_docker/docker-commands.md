# Docker Commands Reference 🧾  

### 🧱 Image Management
```bash
docker pull <image>
```
### 🐳 Container Lifecycle
```bash
docker run -d -p <host_port>:<container_port> <image>
docker ps
docker stop <container_id>
docker rm <container_id>
```
### 🧩 Compose & Multi-Container
```bash
docker compose ps
docker compose watch
```
